## ✨ Features

- **Moderation** — Ban, kick, mute, timeout, warn, jail, and anti-nuke protection
- **Automod** — Anti-spam, anti-caps, anti-link, anti-invite, anti-mass-mention, emoji spam filtering
- **Logging** — Full audit logs for server events
- **Welcome & Farewell** — Customizable welcome cards and goodbye messages
- **Leveling** — XP and level-up system with leaderboards
- **Giveaways** — Create and manage server giveaways
- **Tickets** — Support ticket system
- **Invite Tracker** — Track who invited who to your server
- **Voice Master** — Dynamic voice channel management
- **Reaction Roles** — Assign roles via reactions
- **Auto-Role** — Automatically assign roles on join
- **Auto-Responder** — Keyword-triggered responses
- **AI Chatbot** — Conversational AI powered by OpenAI
- **Image Generation** — AI image generation via Prodia
- **Games** — Chess, Wordle, Battleship, Connect Four, 2048, Tic-Tac-Toe, and more
- **Fun** — Slots, Blackjack, Country Guess, Type Racer, and more
- **Sentinel** — Advanced server security system
- **Premium** — Exclusive premium features for supporters
